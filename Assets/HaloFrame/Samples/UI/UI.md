# UI框架
## UI层级管理
方案一：在一个容器中管理全屏UI和非全屏UI
方案二：在两个容器中管理全屏UI和非全屏UI，非全屏遮盖不处理显示隐藏
区分全屏与非全屏，为了让被全屏覆盖的UI可以隐藏，减少drawcall。

方案二：
打开顺序：
1(全屏) 2 3(全屏) 4

当前状态：
全屏 1（隐藏） 3（显示）
非全屏 2（显示） 4（显示）

4先关闭，不会触发其他界面的显示隐藏。1(隐藏) 2（显示） 3（显示）
3先关闭，1 （显示） 2（显示） 4（显示）
